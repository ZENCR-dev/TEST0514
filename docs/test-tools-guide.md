# 测试工具使用指南

本文档介绍如何使用测试工具进行前后端联调和角色模拟。

## 快速启用

在浏览器控制台中运行以下命令，启用所有测试工具：

```javascript
import('@/scripts/enableTestTools').then(module => console.log('测试工具已启用'));
```

刷新页面后，测试面板将显示在页面右上角。

## 测试面板功能

测试面板包含两个主要功能：

### 1. 环境切换

- **联调环境**：连接到NestJS后端（端口3001）
- **Mock环境**：使用Next.js API Routes（端口3000）

切换环境后，页面将自动刷新以应用新的API环境。

### 2. 角色模拟

支持模拟以下角色：

- **管理员**（admin）
- **医师**（practitioner）
- **药房**（pharmacy_operator）
- **患者**（patient）

切换角色后，页面将自动刷新以应用新的角色。

## 处方保存功能

处方保存功能已经实现，现在可以将处方数据保存到后端数据库。使用步骤：

1. 确保环境切换器设置为"联调环境"
2. 创建处方并填写必要信息
3. 在预览页面点击"保存到系统"按钮

## 常见问题

### 无法连接到后端API

如果测试面板显示"API离线"，请检查：

1. 后端服务是否已启动（端口3001）
2. 是否有网络连接问题
3. 是否有CORS配置问题

### 角色切换不生效

如果角色切换不生效，请检查：

1. 是否在开发环境中运行
2. 浏览器控制台是否有错误信息
3. 尝试清除localStorage并重新启用测试工具

## 禁用测试工具

在浏览器控制台中运行以下命令，禁用所有测试工具：

```javascript
import('@/scripts/enableTestTools').then(module => module.disableAllTestTools());
```

刷新页面后，测试工具将被禁用。
